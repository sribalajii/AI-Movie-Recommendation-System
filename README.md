# MoodMovie - Movie Recommendation App

A smart movie recommendation app that suggests movies based on your mood, with special features like Eating Mode and Surprise Me.

## Features

### 🎯 Core Features
- **Mood-based recommendations** (Happy, Sad, Tired, Angry, Romantic)
- **Quick Pick** for instant decisions
- **Surprise Me** button for random suggestions
- **Eating Mode** for light entertainment while eating
- **Time-based auto suggestions**
- **Bilingual support** (English & Tamil)

### 🎨 UI/UX Features
- Modern glassmorphism design
- Dark/Light mode toggle
- Mobile-first responsive design
- Smooth animations and transitions
- Swipeable movie cards
- Gradient background animation

### 📱 PWA Features
- Installable as app
- Works offline
- Fast loading
- App-like experience
